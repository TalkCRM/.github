# TalkCRM

**병원·의료기관을 위한 CRM과 AI 상담 에이전트를 만듭니다.**

TalkCRM은 성민네트웍스(Sungmin Networks)가 운영하는 제품 조직입니다. 의료기관이 환자·고객과 나누는 모든 대화를 한 곳에서 관리하는 옴니채널 CRM과, 그 위에서 실제 상담을 수행하는 생성형 AI 에이전트를 함께 개발합니다.

---

## 우리가 만드는 것

| 영역 | 내용 |
|---|---|
| **옴니채널 CRM** | 카카오톡·문자·웹챗 등 여러 채널의 고객 문의를 하나의 관리자 콘솔에서 통합 응대 |
| **AI 상담 에이전트** | 예약 문의, 진료 안내, 병원 지식베이스 기반 응답을 자동으로 처리하는 생성형 AI |
| **지식관리(KMS)** | 병원별 진료 정보·FAQ를 구조화해 AI 응답의 근거로 활용하는 지식베이스 |
| **예약·상담 자동화** | 상담 흐름과 예약 프로세스를 대화형으로 이어주는 워크플로 |

여러 의료기관에 맞춰 적용하되, 하나의 제품 구조로 수렴시키려 합니다.

---

## 기술 지향점

- 클라우드 리전 내 생성형 AI(AWS Bedrock)를 활용한 한국어 상담 품질 확보
- 채널이 늘어나도 무너지지 않는 옴니채널 통합 아키텍처
- 의료 도메인 데이터를 다루는 만큼 보안·개인정보 보호를 우선하는 설계
